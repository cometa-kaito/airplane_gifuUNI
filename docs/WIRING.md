# 配線ガイド

## 全体配線図

```
                    +---------------- 機体側 -------------------+
                    |                                           |
                    |  [nRF52840 Sense]                         |
                    |   USB -- (給電・デバッグ用)                |
                    |   D0 --- サーボ0 信号 (右エルロン)          |
                    |   D1 --- サーボ1 信号 (左エルロン)          |
                    |   D2 --- サーボ2 信号 (エレベータ)          |
                    |   D6 --- TX -----+                        |
                    |   D7 --- RX -----+--+                     |
                    |   GND --- 共通    |  |                    |
                    |                  ↓  ↓                    |
                    |   [ESP32-C3 #2 機体側]                    |
                    |   D7 (RX, GPIO20) ←--+                    |
                    |   D6 (TX, GPIO21) ---+                    |
                    |   USB or 5V (給電)                         |
                    +-------------------------------------------+
                                       ↕  無線 ESP-NOW
                    +---------------- 地上側 -------------------+
                    |  [ESP32-C3 #1 地上側]                     |
                    |   USB --- PC（電源・データ）              |
                    +-------------------------------------------+
                                                ↕
                                              [PC]
                                              シリアルモニタ
                                              Python ビューア
```

## Serial1 (UART) クロス配線

| nRF52840 Sense | 接続 | ESP32-C3 #2 |
|---|---|---|
| D6 (TX) | ←→（クロス） | D7 (RX, GPIO20) |
| D7 (RX) | ←→（クロス） | D6 (TX, GPIO21) |
| GND | 共通 | GND |

⚠️ **TX 同士、RX 同士をつなぐと通信できません**。必ず片方の TX を相手の RX に、その逆も同様に。

## サーボ配線

サーボの線は通常 3 本：

| 線色 | 信号 | 接続先 |
|---|---|---|
| 茶（または黒） | GND | nRF52840 GND（外部電源使用時は外部電源 GND と共通化） |
| 赤 | +5V（VCC） | nRF52840 5V または外部電源 |
| 橙（または黄） | 信号 | nRF52840 D0（右エルロン）/ D1（左エルロン）/ D2（エレベータ） |

### サーボ電源について
- **USB 給電**：軽量サーボ 1 個ならOK、複数同時駆動だと足りない
- **外部電源推奨**：4.8〜6V 程度のバッテリーまたはレギュレータ出力
- **GND 必ず共通化**：信号線が動作するためには電源系と論理系の GND を結ぶ

## 電源系（飛行時）

```
[2セル Li-Po 7.4V] ----+---- L7805 (3端子レギュレータ) ---- 5V ---- nRF52840 / ESP32 / サーボ
                       |                                   GND
                       +---- 共通 GND
```

⚠️ Li-Po バッテリーは充放電仕様を守ること（過放電は発火・破損の原因）。

## 機体・地上の電源パターン

| 用途 | 機体側 | 地上側 |
|---|---|---|
| 開発・デバッグ時 | nRF/ESP32 を別々の USB | ESP32 を PC USB |
| 飛行時 | Li-Po + レギュレータ（USB 不要） | ESP32 を PC USB |

## 配線チェックリスト

- [ ] Serial1 が **TX↔RX クロス**になっている
- [ ] 全ボードの **GND が共通**
- [ ] サーボ信号線が **D0/D1/D2** に正しい順序で
- [ ] サーボ電源が **5V**（3.3V では動かない）
- [ ] バッテリー使用時は **電圧が定格内**
